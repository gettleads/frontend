# GettLeads Frontend

A static frontend for [GettLeads](https://github.com/gettleads) — a B2B contact intelligence platform that helps sales teams find verified emails and phone numbers from 850M+ LinkedIn profiles.

## Pages

| File | Description |
|------|-------------|
| `FE Project/home.html` | Landing page with hero, features, and CTA |
| `FE Project/login.html` | Login and sign up forms |
| `FE Project/pricing.html` | Pricing plans with monthly/annual toggle |
| `FE Project/linkedin-people-database.html` | LinkedIn People Database dataset page |
| `FE Project/footer.html` | Standalone footer component |
| `CSS Cheatsheat/index.html` | CSS reference cheatsheet |

## Running Locally

No build step needed. Just open any `.html` file directly in your browser:

```bash
open "FE Project/home.html"
```

Or use the VS Code Live Server extension for auto-reload on save.

## Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- [Inter](https://fonts.google.com/specimen/Inter) via Google Fonts
- [Font Awesome 6](https://fontawesome.com/) for icons
